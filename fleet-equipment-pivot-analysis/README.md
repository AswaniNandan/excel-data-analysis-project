# Fleet Equipment Pivot Analysis

Project Overview
  This project focuses on analyzing fleet equipment inventory data using Pivot Tables in Microsoft Excel.

- Dataset
The dataset used in this project is the cleaned fleet equipment data, which includes:
   - Department
   - Equipment Class
   - Equipment Count

- Analysis Performed
   - Equipment count by department
   - Equipment distribution by equipment class
   - Identification of departments with highest equipment count

- Pivot Tables Created
   - Department vs Equipment Count
   - Equipment Class vs Equipment Count

- Tools Used
   - Microsoft Excel
   - Pivot Tables

Key Insights
  - Departments with the highest equipment count were identified
  - Certain equipment classes contribute more to overall inventory
  - Equipment distribution varies across departments
